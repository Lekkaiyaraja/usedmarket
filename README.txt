UsedMarket - Static Classifieds App
==================================

This is a static single-page web application that demonstrates a simple classifieds UI with sample products.
Features:
- Top navbar with Buy and Sell buttons.
- Sell modal form to add products (name, description, price, category).
- Popup confirmation messages.
- 20 seeded sample electronics products (no images).
- Buy button marks product as SOLD and persists to localStorage.
- Filters: search, category, sort.
- Everything is client-side, no backend required.

To run:
1. Unzip the archive and open `index.html` in your browser.
2. The app stores products in browser localStorage. Clear localStorage to reset seed data.

Files included:
- index.html
- styles.css
- app.js
